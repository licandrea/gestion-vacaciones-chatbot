# gestion-vacaciones-chatbot
Descripción

Este proyecto simula un chatbot para la gestión de solicitudes de vacaciones de los empleados de TechLogic S.A.

El sistema permite:

Ingresar el legajo del empleado.
Ingresar la cantidad de días solicitados.
Verificar la disponibilidad de días de vacaciones.
Aprobar o rechazar la solicitud según las reglas definidas.
Diagrama BPMN

El repositorio incluye el diagrama BPMN del proceso propuesto en el archivo ultima.png.

Lógica de decisión

La decisión principal del sistema se implementa mediante estructuras condicionales (if/else).

Si el empleado posee días suficientes, la solicitud es aprobada.
Si no posee días suficientes, la solicitud es rechazada.
